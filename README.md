# Personal Portfolio Website

A responsive personal portfolio website built using HTML, Tailwind CSS, and JavaScript. This project showcases my skills, projects, and contact information in a clean and modern design.

## Live Site
site link = https://rasel-portfolio-1.netlify.app/

## Features

- Responsive design for all devices
- Smooth scrolling navigation
- Typing animation
- Skills section
- Project showcase
- Contact form with EmailJS integration
- Social media links
- Dynamic footer year
